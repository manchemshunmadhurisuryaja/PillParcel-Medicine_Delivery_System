# 💊 PillParcel: Medicine Delivery System

## 📌 Introduction

PillParcel is an online platform developed to simplify the process of delivering medicines to students within a specific college campus. It creates a smooth connection between two main users — students and sellers. Students can upload their prescriptions and choose a medical shop, while sellers can manage these requests and deliver the required medicines efficiently.

---

## ✨ Key Features

* Easy submission and management of prescriptions
* Efficient medicine delivery within the campus
* Option for students to select their preferred medical store

---

## 👥 User Roles and Functionalities

### 🔹 Sellers

Sellers can register and log in to the system to handle orders. They can view prescriptions submitted by students, process them, and arrange delivery to the campus. Notifications are sent whenever a new order is placed, helping sellers respond quickly.

### 🔹 Students

Students can sign up or log in to the platform to access services. They can upload prescriptions with necessary details and choose a medical shop for order fulfillment.

---

## 🗂️ Project Structure

The complete system is implemented in a single Python file named **parcel.py**, which contains all the functionalities required to run the application.

---

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone <repository_url>
```

2. Navigate to the project folder

```bash
cd PillParcel
```

3. Run the application

```bash
python parcel.py
```

---

## 🎯 Conclusion

PillParcel provides a simple and efficient solution for medicine delivery within a college campus. It ensures that students can easily access medicines while enabling sellers to manage and fulfill orders effectively.
